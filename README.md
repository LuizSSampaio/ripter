# Ripter

A simple rust encrypter and decrypter cli.

## Run Locally

Clone the project

```bash
  git clone https://github.com/LuizSSampaio/ripter.git
```

Go to the project directory

```bash
  cd ripter
```

Run the command

```bash
  cargo run -- [files]
```
